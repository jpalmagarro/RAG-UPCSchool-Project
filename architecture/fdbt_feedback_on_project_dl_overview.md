## Feedback del documento Project_DL_Overview

Utilizar los términos _Encoder_ y _Decoder_ para referirse a las dos partes del sistema puede introducir cierta confusión conceptual, ya que sugiere la idea de un único modelo LLM de tipo encoder-decoder, propio de la arquitectura original de Transformer. Sin embargo, nuestro sistema no se construye bajo esa división, sino que se aborda como un sistema RAG estructurado en módulos funcionales claramente diferenciados.

## Propuesta de nombre y descripción cortos del sistema.

**COSORA** stands for _COnstruction Site ORAcle_.
 
This system has the following descripción:  LLM-based system for answering queries using context from construction site inspection reports.
